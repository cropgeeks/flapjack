Download Flapjack

The most recent release of Flapjack is 1.12.05.25 (25th May 2012).

Please use the links below to download the Flapjack installer most suitable for your operating system. Flapjack is currently available for:

    Windows (32 bit) or Windows (64 bit)
    Linux (32 bit) or Linux (64 bit)
    Apple Mac OS X (Java 6 required)
    Solaris (Sparc)

A Java Web Start version of Flapjack also exists, and can be made available on request in certain circumstances, however this is not something we can officially support.

Follow the instructions provided by the graphical installer to install Flapjack. Note that Flapjack should not require administrator privileges to install or run, but if you have any problems, don't hesitate to contact us at flapjack@hutton.ac.uk.

Flapjack is developed on Windows 7, and gets additional regular testing on Mac OS X 10.7 and Centos 6.

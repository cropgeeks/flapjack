File Menu

This reference page describes each item found within the File menu of Flapjack's main menu bar.

New project

Creates a new Flapjack project.

Open project

Displays the Open Project dialog, which prompts you for a previously created project located on disk, then opens it within Flapjack.

Save project

Saves the current project to disk, optionally prompting you for a filename if the current project has not been saved yet.

Save project as

Displays the Save Project dialog, prompting you for a filename to save the current project as, then saves it.

Import data

Displays the Import Data Options dialog, which allows you to import map and genotype data into Flapjack in a variety of different ways.

Recent projects

Opens a sub-menu of recently accessed project files. Select any of the options to open that project within Flapjack.

Exit

Closes Flapjack. Note that on Mac OS X, this option is remapped to Flapjack > Quit on the system menu.

Download Flapjack

The most recent release of Flapjack is 0.10.05.07 (7th May 2010).

Please use the links below to download the Flapjack installer most suitable for your operating system. Flapjack is currently available for:

    Microsoft Windows (19.1MB)
    Apple Mac OS X (2.8MB)
    Linux (22.0MB)

Follow the instructions provided by the graphical installer to install Flapjack. Note that Flapjack should not require administrator privileges to install or run, but if you have any problems, don't hesitate to contact us at flapjack@scri.ac.uk.

Flapjack is developed on Windows 7, and gets additional regular testing on Windows XP SP3, OS X 10.6, Ubuntu Linux, and Centos 5.

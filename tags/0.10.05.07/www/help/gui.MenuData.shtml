Data Menu

This reference page describes each item found within the Data menu of Flapjack's main menu bar.

Sort lines

Opens a sub-menu displaying various options for sorting the lines into a new order.

By similarity

Displays the Sort Lines By Similarity dialog that allows you to order the lines by their similarity to a selected comparison lines, based upon the currently selected markers across any subset of chromosomes.

By trait

Displays the Sort Lines By Trait dialog that allows you to order the lines by sorting them on up to three selected traits.

Filter QTLs...

Displays the Filter QTLs dialog that allows you to determine which QTLs will be visible based on their trait and experiment values.

Find by name

Displays the Find By Name dialog that allows you to search for either lines or markers using standard or regular-expression search strings.

Data summary

Displays the Data Summary dialog that provides a summary overview of all alleles across all chromosomes.

Query database

Opens a sub-menu displaying options for searching and displaying results from remote databases from within Flapjack.

Line information

Sends the name of the line under the mouse (at the time the menu was displayed) to a remote database for further information.

Marker information

Sends the name of the marker under the mouse (at the time the menu was displayed) to a remote database for further information.

Settings

Displays the Database Link Settings dialog that allows you to set or update the URLs that Flapjack will use when attempting to query a remote database.

Rename data set

Displays the Rename dialog that allows you to rename the current data set.

Remove from project

Offers the option to permamently delete the current data set from the active project.

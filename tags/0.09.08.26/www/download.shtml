Download Flapjack

The most recent release of Flapjack is 0.09.08.28 (28th August 2009).

Please use the links below to download the Flapjack installer most suitable for your operating system. Flapjack is currently available for:

    Microsoft Windows (19.0MB)
    Apple Mac OS X (2.8MB)
    Linux (19.1MB)
    Solaris (sparc) (26.3MB)

Follow the instructions provided by the graphical installer to install Flapjack. Note that Flapjack should not require administrator privileges to install or run, but if you have any problems, don't hesitate to contact us at flapjack@germinate.org.uk.

Flapjack is developed on Windows XP SP3 (32bit), and gets additional regular testing on OS X 10.5 (64bit), Ubuntu Linux (32bit), and Centos 5 (32bit).

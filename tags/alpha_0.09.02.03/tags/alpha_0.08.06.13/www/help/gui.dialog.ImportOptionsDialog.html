Import Data Options

The Import Data Options dialog is used to select which method you'd like to use to import a data set into Flapjack.

The following options are available:

  • Providing the map and genotype data from files located on disk
    Selecting this method will open the Import Data dialog and allow you to choose a file containing map data and a file containing genotype data to import into Flapjack.

  • Connecting to a Germinate database and importing directly
    This option is not yet implemented.

  • Using the built-in example dataset bundled with Flapjack
    Selecting this method will to open an examplar dataset that is suitable for use while getting to know Flapjack.

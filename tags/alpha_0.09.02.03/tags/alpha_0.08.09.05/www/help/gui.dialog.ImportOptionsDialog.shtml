Import Data Options

The Import Data Options dialog is used to select which method you'd like to use to import a data set into Flapjack.

The following primary options are available:

  • Providing the map and genotype data from files located on disk
    Selecting this method will open the Import Data dialog and allow you to choose a file containing map data and a file containing genotype data to import into Flapjack.

  • Connecting to a Germinate database and importing directly
    This is a placeholder option for possible future inclusion.

  • Using the built-in example data set bundled with Flapjack
    Selecting this method will open an examplar data set that is suitable for use while getting to know Flapjack.

Secondary options are also available, once a primary data set has been loaded:

  • Import trait information from a file located on disk
    Selecting this option prompts you to provide a file containing trait information that can be associated with the lines contained in the main data set.

  • Import QTL information from a file located on disk
    This is a placeholder option for possible future inclusion.

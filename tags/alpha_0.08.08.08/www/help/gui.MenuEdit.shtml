Edit Menu

This reference page describes each item found within the Edit menu of Flapjack's main menu bar.

Undo

Undoes the last operation (if that operation is supported by the undo/redo system).

Redo

Redoes the last operation (if that operation is supported by the undo/redo system).

Navigation mode

Switches the Flapjack interface to Navigation Mode, meaning that most mouse interactive events upon the main canvas will affect navigation.

Marker mode

Switches the Flapjack interface to Marker Mode, meaning that most mouse interactive events upon the main canvas will affect markers.

Line mode

Switches the Flapjack interface to Line Mode, meaning that most mouse interactive events upon the main canvas will affect lines.

Select markers

Opens a sub-menu displaying various options for selecting markers. Choosing any of these options will also switch Flapjack to Marker Mode.

Select all

Selects all of the markers within the current chromosome.

Select none

Selects none of the markers within the current chromosome (deselects all).

Invert

Inverts the selection of markers within the current chromosome. Markers that are selected are deselected, and markers that are not selected become selected.

Show/hide markers

Displays the Show/Hide Markers dialog that allows you to hide certain markers from being displayed, or to restore previously hidden markers.

Select lines

Opens a sub-menu displaying various options for selecting lines. Choosing any of these options will also switch Flapjack to Line Mode.

Select all

Selects all of the lines within the current view.

Select none

Selects none of the lines within the current view (deselects all).

Invert

Inverts the selection of lines within the current view. Lines that are selected are deselected, and lines that are not selected become selected.

Show/hide lines

Displays the Show/Hide Lines dialog that allows you to hide certain lines from being displayed, or to restore previously hidden lines.

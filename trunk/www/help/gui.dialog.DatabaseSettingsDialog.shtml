Database Link Settings

Use the Database Link Settings dialog to set or update the URLs that Flapjack will use when attempting to query a remote database for line or marker information.

  • Line searches
    Enter a valid URL containing a location that can be queried for information on a line, where $LINE will be replaced by the actual name of the line.
  • Marker searches
    Enter a valid URL containing a location that can be queried for information on a marker, where $MARKER will be replaced by the actual name of the line.

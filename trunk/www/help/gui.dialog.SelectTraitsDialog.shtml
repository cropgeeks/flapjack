Select Traits

Use the Select Traits dialog to pick which traits should be displayed on the traits heatmap alongside the list of line names.

Up to three traits can be selected by choosing each trait's name using the drop down lists. To not select a trait for display in a particular column, select the blank option at the top of that list.

Trait 1 corresponds to the left-hand side column of the heatmap, Trait 2 to the middle column, and Trait 3 to the right-hand side column.

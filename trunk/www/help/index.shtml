Welcome to Flapjack Help

Help documentation for Flapjack is currently in progress - many topics are still not covered at this time.

To get help with using Flapjack, select Help > Help contents from the menu bar, click the Help toolbar button, or press F1 at any time. Help is also available in many of Flapjack's dialogs by clicking the Help button if visible.

Note that Flapjack Help is only available while Flapjack is running on a computer with an active internet connection. Help is also only available in English (UK), regardless of the language selected within Flapjack.

If you cannot find what you are looking for using the online help, then you can get more help and support by emailing flapjack@germinate.org.uk.

Help Menu

This reference page describes each item found within the Help menu of Flapjack's main menu bar.

Help contents

Opens your default web browser and takes you to this online help.

License

Opens your default web browser and takes you to the Flapjack Software Licence online help page.

Preferences

Displays the Preferences dialog that allows you to change several options that affect the way Flapjack is used and is displayed. Note that on Mac OS X, this option is remapped to Flapjack > Preferences on the system menu.

Check for update

Displays the Updater dialog that will connect to the internet and check to see if a new version of Flapjack is available. If it is, you will be prompted to download and install it. You should save any open projects first.

About Flapjack

Displays the About Flapjack dialog that shows information on the current version, copyright information, and the list of authors/contributors. Note that on Mac OS X, this option is remapped to Flapjack > About Flapjack on the system menu.
